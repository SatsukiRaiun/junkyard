## やること
1. HTML3 → Markdown への変換
2. Shift JIS → UTF-8 への変換
3. ポストエディットと改訳

### sed スクリプト

これで前述の 1, 2 は済むが、よけいなパイプ記号とホワイトスペースだらけの表組みになっているので出力はプレーンテキストにしてからマークアップ打ち直しやポストエディットしたほうが楽かと。

``` bash
for f in *.htm; do iconv -f SHIFT_JIS -t UTF-8 "$f" | pandoc -f html -t markdown -o "${f%.htm}.md"; done
```

rtrim 相当
``` bash
sed 's/[[:space:]]*$//' file.txt
```

ゆっくり進めていきます。
