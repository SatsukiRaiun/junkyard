現時点では再開未定です。ほぼ後継言語は Phix なのでバックポート以外では使う機会はほとんど無いでしょう。

## Euphoria 3.x (eu3 ディレクトリ)
 * 当時の環境 (Windows 95, 98, Me, NT 4.0, 2000, XP)、または DOS32 版は DOSBOX で使用してください (PC-9801, FMR/FM−TOWNS, X68000 など旧型国産機では動作しません)。 OpenWatcom C++ などのバージョンや設定を合わせることで動作はしますが Vista 以降は正式サポートされていません。
 * 便宜上、著作権表示は当時のままにしてあります。連絡先など変わっていますが翻訳者は自称同一人物です。

### 桜月ヲンラヰン (旧OSDN/現Junkyard) - 現行版 (2026年版)
 * Markdown, Sphinx, UTF-8, TextTra によるポストエディットと改訳など主に近代化改修作業です。

## FAQ
 * Q1. ツインテールの開発再開予定はありますか？\
   A1. さて、なんのことでしょうか？ 縦ロールですか、サイババロールですか？ わかりません。こうですか？ はっきり言いますが、あの時代を逃した時点でおそらく無いと思います。無論、当時の開発機材は未だありますし新たに Termux での開発構築も出来ます。とはいえ、できたとしてもドキュメントまわりの整備や英訳程度です。

 * Q2. なぜ、いまさら改修なんて？\
   A2. Seed7 の翻訳でコケたからです。それがなければ Phix に手を出そうなんて考えもしなかったです。試作言語として使えるものが必要なので、それまでのつなぎです。最終的に Pharo へ行きます。

 * Q4. Euphoria 3.x の開発再開はありますか？\
   A4. 歴史的役割は追えています。Phixをお使いになるか必要ならご自身でどうぞ。一応、2022年頃に FreeDOS向けに改良された v3.2.1 は公開されています。

 * Q5. DOS32 版での日本語表示と入力したい。\
   A5. 無変更ではできません。字句解析器の改造などマルチバイト文字列処理ルーチン、ディレクトリパス問題の解決、フォントレンダラー、テキスト表示、単文節入力ライブラリを作りましょう。

 * Q6. Shian Lee氏 が運営していた rapideuphoria311 のサイトってどうなったの？\
  A7. ドメインごと消滅したようです。 Lib2 と edx は Internet Archive から未だ[入手できる](https://web.archive.org/web/20210509004352/https://www.rapideuphoria311.com/#lib2)ようですが。
